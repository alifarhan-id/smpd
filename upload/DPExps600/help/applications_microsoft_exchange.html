Supported Platforms

The Data Protector Express Agent for Microsoft Exchange supports backup and restore operations for Microsoft Exchange 2000 Server, Microsoft Exchange Server 2003, Microsoft Exchange Server 2007, Microsoft Exchange Server 2010 and restoring from Microsoft Exchange Server 5.5 (if backed up with a previous version of Data Protector Express)

Some editions of Data Protector Express do not provide a license to backup Exchange data. By default, an evaluation version of the Data Protector Express agent for Microsoft Exchange is installed automatically when you install Data Protector Express on a Windows server machine that is running Microsoft Exchange. You can use this evaluation license for 60 days. To continue using the agent, contact your sales representative to purchase an edition of Data Protector Expressthat supports Application agents.
